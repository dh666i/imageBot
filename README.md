# ImageBot 图片生成工具

在 Windows 电脑上输入文字或上传参考图片，即可生成或修改图片。

[下载最新版](https://github.com/dh666i/imageBot/releases/latest)

## 开始使用

1. 下载最新版压缩包。
2. 把压缩包完整解压到一个文件夹。
3. 双击 `ImageBot.exe`。
4. 第一次打开时填写接口密钥，点击 `保存`。
5. 在页面底部输入想生成的画面，点击 `生成`。

接口地址和模型已有默认值，一般不需要修改。

程序启动后会留在 Windows 右下角托盘。双击托盘图标可重新打开页面，右键可以重新启动或退出。

如果 `ImageBot.exe` 无法运行，可以双击 `启动图片WebUI-独立Config版.bat` 备用启动。

## 生成图片

### 文字生成图片

直接输入想要的画面，例如：

```text
一张黑白高级感的咖啡机产品海报，干净背景，真实摄影
```

### 上传图片修改

点击 `上传图片`，或把图片拖进输入框，再描述想怎么修改。例如：

```text
把背景换成明亮的咖啡店，主体保持不变
```

也可以复制图片，然后直接粘贴到输入框。

## 图片和历史

生成的图片保存在当前电脑。打开右上角 `设置`，进入 `本地文件`，可以查看占用空间并清理图片、历史记录或日志。

每张生成结果都可以下载、查看大图、继续修改或再次生成。

## 更新和恢复

打开 `设置`，展开 `维护工具`，点击 `检查更新`。发现新版后点击 `立即更新`，程序会自动下载并校验文件。

更新不会删除接口配置、历史记录、日志或已生成图片。更新后如果出现问题，回到 `维护工具`，点击 `恢复上一版本`，然后重新启动 ImageBot。

从 `v1.3.0` 页面内更新时，旧版更新器不会下载首次新增的 `ImageBot.exe`。需要无黑窗启动时，请手动下载最新版压缩包并重新解压，原来的 `config.ini` 可以继续使用。

## 常见问题

### 双击后没有打开网页

先确认压缩包已经完整解压，再双击 `ImageBot.exe`。也可以双击托盘图标，或在浏览器打开：

```text
http://127.0.0.1:7861
```

### Windows 阻止运行

当前发布包可能没有 Windows 数字签名。确认文件来自本项目发布页后，可以在系统提示中点击 `更多信息`，再点击 `仍要运行`。

### 提示接口密钥缺失

打开右上角 `设置`，填写接口服务提供方给你的密钥，然后点击 `保存`。

### 生成时间较长

图片生成通常需要几十秒。网络或接口服务繁忙时可能更久，本工具默认等待 240 秒。

### 想先体验界面

打开 `设置`，展开 `高级设置`，开启 `演示模式`。演示模式不会请求真实接口，也不会消耗额度。

## 说明

- 支持 Windows 10 和 Windows 11。
- 接口密钥只保存在使用本工具的电脑上。
- 图片生成费用和接口稳定性由接口服务提供方决定。
