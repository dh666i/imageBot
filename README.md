# ImageBot 图片生成工具

在 Windows 电脑上，输入文字或上传图片，就能生成图片或修改图片。

[下载最新版](https://github.com/dh666i/imageBot/releases/latest)

## 开始使用

1. 下载最新版压缩包。
2. 解压到电脑上的任意位置。
3. 双击 `启动图片WebUI-独立Config版.bat`。
4. 第一次打开时填写接口密钥，点击保存。
5. 在页面底部输入内容，点击 `生成`。

接口地址和模型已经有默认值，一般不需要修改。

## 生成图片

### 文字生成图片

在输入框描述想要的画面，例如：

```text
一张黑白高级感的咖啡机产品海报，干净背景，真实摄影
```

### 上传图片修改

点击 `上传图片`，或把图片拖进输入框，然后描述想怎么修改。

```text
把背景换成明亮的咖啡店，主体保持不变
```

也可以直接复制图片后粘贴到输入框。

## 生成完成后

每张图片都可以：

- 下载到电脑
- 查看大图
- 继续修改
- 使用相同提示词再次生成

页面底部可以快速选择方形、横图或竖图。其他选项在 `更多设置` 中。

## 图片和历史

生成的图片会保存在本地，不会自动上传到本项目。

打开右上角 `设置`，进入 `本地文件`，可以查看占用空间，也可以分别清理图片、历史记录和日志。

## 检查更新

打开 `设置`，展开 `维护工具`，点击 `检查更新`。

如果有新版，页面会显示 `有可用更新`，点击 `立即更新` 即可。更新不会删除已经保存的配置、图片和日志。

## 常见问题

### 双击后没有打开网页

确认压缩包已经完整解压，再重新双击启动文件。也可以手动打开：

```text
http://127.0.0.1:7861
```

### 提示接口密钥缺失

打开右上角 `设置`，填写接口密钥并点击 `保存`。接口密钥由你使用的接口服务提供。

### 生成时间比较长

图片生成通常需要几十秒，请等待页面显示结果。网络或接口服务繁忙时，可能需要更长时间。

### 想先体验界面

打开 `设置`，展开 `高级设置`，开启 `演示模式`。演示模式不会调用真实接口，也不会消耗额度。

## 说明

- 本工具适用于 Windows 电脑。
- 接口密钥只保存在使用本工具的电脑上，不会上传到本项目。
- 图片生成是否收费、收费标准和服务稳定性由接口服务提供方决定。
